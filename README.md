This repository managed by NORC at the University of Chicago documents processing and analytic steps for producing weighted chronic disease estimates in MENDS. NORC (the weighting and estimation vendor supporting MENDS) releases annotated code files at various geographic levels (national, state, county, ZIP code) for priority MENDS indicators. The repository contains code files for hypertension and hypertension control indicators and evolves to align with weighting and modeling methodology updates for MENDS (SAS/R). Similar code files for other MENDS indicators are available by request.
